# Vue-Spring Template

1. git clone git@github.com:Lorenayannnnn/VueSpringTemplate.git
2. frontend:
	1. cd frontend
	2. npm install
	3. directory structure:

		2. components

		4. config: support axios for sending requests
		5. router
3. backend: Java-spring framework
